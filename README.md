# naatai

Crypto futures trading workspace.

The trading-signal engine lives in the companion repo **[`naatai-ai`](https://github.com/robintex7-cpu/naatai-ai)**:
a Wyckoff-driven scanner over the full Binance USDⓈ-M perpetual universe that
detects pairs coming in / going out and generates short-term and long-term
LONG/SHORT signals from low-lag indicators, Wyckoff structure, and
futures-only data (open interest, funding, long/short ratio).

See `naatai-ai/README.md` for setup, usage, and methodology.

> ⚠️ Not financial advice. Leveraged perpetual futures are high-risk.
